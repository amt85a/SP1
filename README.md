Ceci est une application web utilisée dans le cadre des épreuves du BTS SIO. Elle répond à des besoins énoncés dans un scénario fictif. Il ne s'agit pas d'une application répondant à des besoins réels ou utilisable dans un cadre professionnel.

Cahier des charges de l’application : Application web permettant à un agent Seven de réserver un véhicule pour un trajet (aller simple ou aller retour) pour un client. Cette application est en lien avec une API. On y retrouve les fonctionnalités suivantes : FrontOffice :

F1 : Authentification avec token F2 : Tableau de bord représentant la liste des réservations F3 : Formulaire de réservation d’un véhicule F4 : Profil utilisateur + possibilité de modifier les informations (pas réalisée) BackOffice : F1 : Authentification avec token en tant qu’admin (pas réalisée) F2 : Formulaire permettant de créer un nouveau compte utilisateur, ainsi que la possibilité D’activer/désactiver un utilisateur (pas réalisée) Tests unitaires et fonctionnels :
